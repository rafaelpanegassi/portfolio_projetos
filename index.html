    Pipeline de Dados Utilizando Modern Data Stack

    Este é um projeto End-to-End onde empreguei uma variedade de ferramentas da Data Modern Stack de código aberto.
    Utilizei o Airbyte para a coleta e ingestão de dados, garantindo que informações cruciais fossem capturadas de várias fontes.
    Em seguida, recorri ao Dbt para modelar e transformar esses dados, tornando-os prontos para análise. Como um Data Warehouse confiável, escolhi o Snowflake para armazenar os dados transformados, garantindo sua integridade e acessibilidade.
    A visualização dos insights gerados foi realizada com o auxílio do Metabase, proporcionando uma compreensão visual e significativa dos dados. Para amarrar tudo isso em um pipeline eficiente, o Apache Airflow foi utilizado para orquestrar o fluxo de dados, garantindo um processo completo e coeso.

    Detecção de Fraudes em Cartões de Crédito

    O objetivo deste projeto é desenvolver um modelo de machine learning capaz de identificar potenciais operações fraudulentas de cartões de crédito. O modelo foi treinado em um conjunto de dados de transações, incluindo transações fraudulentas e legítimas.O projeto foi dividido nas seguintes etapas:
    Coleta de dados: Os dados foram coletados de uma base pública do Kaggle.
    Preparação de dados: Os dados foram preparados para análise, removendo erros, limpando-os e transformando-os em um formato consistente.
    Análise exploratória: As análises exploratórias foram realizadas para identificar padrões e tendências nos dados.
    Modelagem de dados: Um modelo de machine learning foi construído para prever se uma transação é fraudulenta ou legítima.O modelo foi avaliado em um conjunto de dados de teste.
    Criação de um app em Streamlit: Um app em Streamlit foi criado para permitir que os usuários simulem transações de cartão de crédito e visualizem a previsão do modelo.
